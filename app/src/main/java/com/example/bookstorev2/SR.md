Обозначение: Приложение магазин книг на языке Kotlin Android Studio.

Описание: Приложение Book Store V2 предлагает пользователю онлайн-магазин книг с возможностью таких функций как:

    Регистрация и логин пользователя, используя пароль и электронную почту с помощью Firebase Authentication.
        Также в будущем можно добавить регистрацию с помощью номера телефона, аккаунта Google.

    Магазин книг будет предлагать большой каталог книг, который будет заполняться вручную администратором приложения в отдельной админ-панеле,
        где последний сможет заполнить необходимую информацию о книге. Сами книги буду представлять из себя обложку,
        название и цену, а экран подробного описания покажет детальное описание книги и другую информацию (дата публикации, автор и пр)

    Должна быть предоставлена возможность фильтрации книг по категориям и по избранным. Добавить в избранное будет возможно с помощью кликабельного значка

    Пользователю должны быть доступны настройки, в которых он сможет изменить язык, тему.

    Пользователь имеет возможность отметить книгу, как прочитанная (которая при отметке добавится в соответствующую категорию)

    Должен быть доступен оффлайн режим, при котором будут отображаться только прочитанные книги

Используемые технологии и темы:
    
    Jetpack Compose
    Coroutines
    основы Clean Architecture + MVVM
    Data base Firebase
    Data base Room
    Hilt/Dagger
    View Model
    Navigation
